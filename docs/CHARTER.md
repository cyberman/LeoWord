# LeoWord Charter

LeoWord is based on AbiWord 2.4.5 Mac/PPC.

Principles:

1. Mac OS X 10.5.8 PowerPC has absolute primacy.
2. Native Cocoa/AppKit behavior wins over portability layers.
3. GTK, GNOME and modern Linux desktop assumptions are out of scope.
4. AbiWord 2.4.5 is the baseline.
5. Later AbiWord versions are used only as controlled diff sources.
6. Every dependency must justify its existence.
7. The first goal is a stable native Leopard word processor, not feature parity with modern AbiWord.
